# Unciv-mod-example

An example mod for [Unciv](https://github.com/yairm210/Unciv), adding a new custom Civilization: **Naianas**.

This mod includes:
- A unique Civilization: **Naianas**, inspired by Mesoamerican cultures (Aztecs, Mayans, Incas)
- A unique religion: **Naianism**
- Two unique buildings: **Sukuia Pyramid** and **Kuiatonus**
- A unique ranged unit: **Phasers**
- Custom dialogues, cities, and visual identity
